API de Medicamentos do Brasil

Dados públicos de medicamentos registrados na ANVISA. Use a API para consultar medicamentos, princípios ativos e situação de registro.

Documentação interativa (Swagger) · ReDoc

Gerar sua API Key

Limite: 5 chaves por hora por IP. Guarde a chave em local seguro; ela não será exibida novamente.

Como usar

Inclua o header em todas as requisições:

X-API-Key: sua-api-key-aqui

Exemplo:

curl -H "X-API-Key: SUA_KEY" https://api.seudominio.com/api/v1/medicamentos
